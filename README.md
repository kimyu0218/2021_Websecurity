### **💻  실행** <br/>

a. Terminal에서 다음을 실행
```shell
$ cd project
$ python3 app.py
```

b. Firefox의 주소창에 다음을 입력
```shell
$ http://127.0.0.1/templates/index.html
```

c. File Upload 버튼을 클릭하여 PDF를 선택하고 Submit 버튼을 클릭하여 해당 PDF 제출
